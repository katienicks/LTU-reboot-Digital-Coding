In today's lesson we first learnt about how Github can be used for single and developer workflows, so that collaborators can work on the same project simultaneously. Seems a bit like when more than one person uses a google doc at the same time ?

In the lesson we learnt about different elements that can be used in HTML

  • Paragraph
  • Heading
  • Link
  • Div
  • List
  • Title

...and 99% of the time these need both an opening tag tag and a closing tag

Other useful HTML to note include

  • < strong > makes text strong < / strong >
  • < em > emphasises text by making italic < / em >

    • ...both of which are really useful to use for accesibility and screenreaders.

    We also discussed Container elements (ones with an opening and closing tag eg. paragraph ) and Stand alone elements (ones without a closing tag, eg < / br >

    Tasks we did involved practising nesting code, understanding attibutes, opening links in new tabs and mailer clients and using images.
